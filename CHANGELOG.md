# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-12-01

### Added
- Initial release of TACO's A CBOR Observer (TACO)
- 358 test cases across core CBOR, Cardano, and edge cases
- Docker-based testing infrastructure
- Support for 18 CBOR libraries across 10 programming languages:
  - Node.js: borc, cbor-x, cbor, cbor-js, cbor-redux, tiny-cbor, cbor-sync, cborg, dag-cbor
  - Python: cbor2
  - Rust: ciborium
  - Go: fxamacker/cbor
  - C: libcbor
  - Java: PeterO.Cbor
  - C#: PeterO.Cbor
  - PHP: spomky-labs/cbor-php
  - Ruby: cbor
  - Perl: CBOR::XS
- HTTP protocol specification for container communication
- Automatic markdown report generation
- Test result rankings and comparison
